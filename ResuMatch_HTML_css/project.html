2. Project Overview
This project aims to build a web-based resume screening application that automates the process of filtering resumes based on job descriptions using Machine Learning (ML) models.
The web app will:

‚úÖ Extract key information from resumes

Recruiters spend hours manually filtering resumes, making the hiring process slow and inefficient. This project aims to automate resume screening, allowing companies to quickly shortlist qualified candidates based on predefined job requirements.
